📊 SuperStore Sales Dashboard with 15 Days Forecast
📌 Project Overview

This project is an interactive Power BI dashboard built using SuperStore sales data.
It analyzes sales performance, profit trends, regional distribution, and provides a 15-day sales forecast using time-series analysis.

🎯 Business Objective

Monitor overall sales and profit performance

Track regional and segment-wise sales distribution

Analyze category and sub-category performance

Predict future sales using forecasting model

📈 Key KPIs

Total Sales: 1.6M

Total Profit: 175K

Total Orders: 22K

Average Ship Days: 4

📊 Dashboard Features
🔹 Sales Analysis Page

Sales by Region

Sales by Segment

Sales by Category & Sub-Category

Monthly Profit Trend

Sales by Ship Mode

Interactive Map Visualization

🔹 Forecast Page

15 Days Sales Forecast

Time Series Trend Analysis

State-wise Sales Comparison
